# mobiquity

# To Run Web Application
1. Navigate to MobiquityWeb folder.
2. from command prompt run "npm i"
3. run "ng serve"
4. launch "http://localhost:4200" from your local browers.


# To Run Unit Test Cases
1. Navigate to MobiquityWeb folder.
2. from command prompt run "npm i"
3. run "npm run test"
